Motionia.js

Built-in ondemand animation library

Motionia.js - a lightweight simplified on demand animation library.With motionia you can animate elements by just one line of code. lightweight, simple fast & easiest to integrate.

Let's animate a box that shows as a strong!

HTML + JS

requestAnimationFrame(anim)
function anim() {
motionia("#target","rippleIn")
requestAnimationFrame(anim)
                }

Let's animate color of a element.

HTML + JS

requestAnimationFrame(color)
function color() {
motionia("#target","color","red","green",
"blue")
requestAnimationFrame(anim)
   }

Lets flip a element

HTML + JS

requestAnimationFrame(flip)
function flip() {
motionia("#target","flipIn")
requestAnimationFrame(flip)
                }

Lets Mix! anims together.

HTML + JS

requestAnimationFrame(mix)
function mix() {
motionia("#target","rotateScaleIn")
requestAnimationFrame(mix)
                }
